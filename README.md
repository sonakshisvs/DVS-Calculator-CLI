## Deployment

Clone the project

```bash
  git clone https://github.com/DVS-Tech-Open-Source-Projects/DVS-Calculator-CLI.git
```

Go to the project directory

```bash
  cd DVS-Calculator-CLI
```

Run .bat File

```bash
    calculator.bat
```
